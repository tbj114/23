# AxaltyX - 专业统计分析软件

## 项目简介

AxaltyX是一款面向学术科研用户的专业统计分析软件，具有功能全面、性能优异、界面友好等特点。通过采用现代化的技术栈和架构设计，AxaltyX能够提供超越SPSS的统计分析能力，同时保持更快的运行速度。

## 核心功能

### 数据管理
- 数据导入：支持CSV、Excel等格式
- 数据编辑：表格编辑器，支持单元格编辑、批量操作
- 数据清洗：缺失值处理、异常值检测、数据转换
- 变量管理：添加、删除、重命名变量，转换变量类型

### 统计分析
- 描述性统计：频数分析、描述性统计量计算、交叉表与卡方检验
- 假设检验：单样本t检验、独立样本t检验、配对样本t检验、方差分析
- 回归分析：多元线性回归、Logistic回归、非线性回归
- 高级统计：因子分析、主成分分析、聚类分析、生存分析

### 绘图功能
- 基础图表：条形图、折线图、散点图、直方图、饼图、箱线图
- 高级图表：热力图、雷达图、3D图表
- 交互式可视化：动态图表、交互式散点图

### 项目管理
- 项目创建、打开、保存
- 分析报告生成：支持HTML格式
- 数据同步：可选字节账号登录同步配置

## 技术栈

- **前端**：Electron.js、HTML、CSS、JavaScript、ECharts
- **后端**：Python、NumPy、SciPy、Pandas、Scikit-learn
- **打包**：Electron Builder

## 系统要求

- Windows 10/11 (32位或64位)
- 4GB以上内存
- 500MB以上磁盘空间
- Python 3.7+

## 安装方法

1. 下载AxaltyX安装包
2. 双击安装包，按照向导完成安装
3. 安装完成后，从开始菜单启动AxaltyX

## 使用指南

### 1. 数据导入
- 点击"导入数据"按钮
- 选择数据文件（CSV、Excel等）
- 预览数据并确认导入

### 2. 数据管理
- 在"数据管理"界面查看和编辑数据
- 使用"变量管理"功能管理变量
- 使用"数据清洗"功能处理缺失值和异常值

### 3. 统计分析
- 在左侧边栏选择分析类型
- 设置分析参数
- 点击"运行分析"按钮
- 查看分析结果

### 4. 绘图
- 在左侧边栏选择"绘图"模块
- 选择图表类型
- 设置图表参数
- 点击"生成图表"按钮
- 查看和导出图表

### 5. 项目管理
- 点击"文件"菜单创建或打开项目
- 保存项目以保留分析结果和图表
- 生成分析报告

## 开发指南

### 环境搭建

1. 克隆项目仓库
2. 安装依赖：
   ```bash
   npm install
   ```
3. 安装Python依赖：
   ```bash
   cd python
   pip install -r requirements.txt
   ```

### 运行开发版本

```bash
npm start
```

### 打包应用

```bash
npm run build
```
