# AxaltyX 产品设计文档

## 1. 产品定位

### 面向用户
- 本科生：学习统计学课程，完成课程作业和学术项目
- 研究生：进行学术研究，分析实验数据，撰写论文
- 科研人员：开展专业研究，进行复杂统计分析
- 问卷与数据分析用户：处理问卷数据，进行市场调研分析

### 产品定位
- **专业统计工具**：功能全面超越 SPSS 标准版
- **性能优势**：速度至少是 SPSS 的 3 倍
- **使用模式**：本地离线优先，可选字节账号登录同步配置
- **设计风格**：现代简洁、字节设计语言（Arco Design / ByteDesign）

### 核心价值
- **专业性**：提供完整的统计分析功能，满足学术科研需求
- **高效性**：优化算法实现，提高数据处理速度
- **易用性**：直观的用户界面，降低学习成本
- **可靠性**：本地离线运行，保证数据安全
- **扩展性**：支持插件系统，方便功能扩展

## 3. 技术栈选择

### 3.1 前端技术
- **框架**：Electron.js（跨平台桌面应用）
- **UI 库**：Arco Design（字节设计语言）
- **图表库**：ECharts、D3.js、Plotly.js
- **数据处理**：Pandas.js、Numeric.js
- **状态管理**：Redux
- **构建工具**：Webpack、Vite

### 3.2 后端技术
- **核心计算**：Python（NumPy、SciPy、Pandas、Scikit-learn、Statsmodels）
- **高性能计算**：Numba、Cython
- **并行计算**：Dask、Ray
- **数据库**：SQLite（本地存储）
- **API**：Flask（用于内部通信）

### 3.3 部署技术
- **打包工具**：Electron Builder
- **安装包**：Windows Installer（.msi）
- **自动更新**：Squirrel.Windows

## 4. 架构设计

### 4.1 系统架构
- **前端层**：Electron 渲染进程，负责用户界面和交互
- **后端层**：Python 计算引擎，负责统计分析和数据处理
- **通信层**：Electron 主进程与 Python 进程间的通信
- **存储层**：本地文件系统和 SQLite 数据库

### 4.2 模块划分
- **数据管理模块**：负责数据的导入、编辑、清洗和管理
- **统计分析模块**：实现各种统计分析方法
- **绘图模块**：负责图表的生成、编辑和导出
- **项目管理模块**：负责项目的保存和加载
- **报告生成模块**：负责分析报告的生成
- **插件管理模块**：负责插件的安装和管理

### 4.3 性能优化
- **计算优化**：使用 Numba 和 Cython 加速计算
- **并行计算**：利用多核心 CPU 进行并行计算
- **内存管理**：优化内存使用，处理大数据集
- **缓存机制**：缓存中间计算结果，提高重复分析的速度

## 5. 用户界面设计

### 5.1 设计原则
- **现代简洁**：采用字节设计语言，界面干净整洁
- **直观易用**：操作流程符合用户习惯，降低学习成本
- **功能完备**：所有统计功能都有相应的界面入口
- **响应式设计**：适应不同屏幕尺寸
- **视觉层次**：通过颜色、空间和排版建立清晰的视觉层次

### 5.2 界面布局
- **顶部导航栏**：包含文件、编辑、视图、分析、绘图、工具、帮助等菜单
- **左侧边栏**：包含数据集管理、变量管理、分析历史等
- **主工作区**：数据表格、分析结果、图表展示等
- **右侧面板**：分析参数设置、图表编辑选项等

### 5.3 交互设计
- **拖拽操作**：支持变量拖拽到分析区域
- **实时预览**：分析结果和图表实时更新
- **上下文菜单**：右键菜单提供常用操作
- **快捷键**：常用操作支持键盘快捷键
- **向导式界面**：复杂分析提供步骤向导

### 5.4 视觉风格
- **色彩方案**：字节设计语言的色彩体系
- **字体**：使用字节推荐的字体
- **图标**：使用字节设计系统的图标
- **动效**：适度的动画效果，提升用户体验
